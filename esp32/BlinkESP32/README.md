# BlinkESP32

Alternates the blue and yellow on-board LEDs of the ESP32 at a 0.5 second interval.

A good first sketch to verify that your Arduino IDE setup and board selection for the ESP32 are working correctly.

## Board selection

In Arduino IDE select **ESP32 Dev Module** as the target board.

## Pins used

| Pin    | LED    |
|--------|--------|
| GPIO32 | Blue   |
| GPIO33 | Yellow |
