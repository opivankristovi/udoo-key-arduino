# BlinkRP2040

Blinks the green on-board LED of the RP2040 on and off at a 0.5 second interval.

This is the "Hello World" of microcontroller programming and a good first sketch to verify that your Arduino IDE setup and board selection are working correctly.

## Board selection

In Arduino IDE select **Raspberry Pi Pico** as the target board.

## Pins used

| Pin    | Function         |
|--------|------------------|
| GPIO25 | Green on-board LED |
